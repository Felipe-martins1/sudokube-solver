Informações Importantes

Quando o visualizador de soluções abrir, é possível, ao clickar sobre um número (quando estiver vendo a solução)
mostrar todos os lugares em que ele aparece, isso ajuda a verificar se as soluções estão de fato corretas

MUITO IMPORTANTE:
Instalar as dependencias do arquivo requirements.txt, utilizando o pipe, gerenciador de dependencias do python
é possível instalar todas rapidamente utilizando o comando: pip install -r requirements.txt
